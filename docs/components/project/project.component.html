ias-icons

ias-icons is an official implementation of the Micro Focus UX icons. This documentation contains a reference implementation and exmaples.

Installation

NPM


                npm install --save @microfocus/ias-icons

Add the stylesheets to your .html file:


                <link rel="stylesheet" href="ias-icons.css" />

Usage

Use the ias-icon element and set it equal to the name of the icon you'd like to use.

Example:


            <ias-icon icon="colors"></ias-icon>
